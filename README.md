I don't just write code. I build automation systems and immersive game mechanics. From deep-level **packet manipulation** to **enterprise Discord architectures**, I focus on creating tools that work efficiently while you sleep.

- 🤖 **Automation:** Building high-performance bots and scripts using **JavaScript/TypeScript** and **Python**.
- 🌐 **Discord Ecosystem:** Crafting secure, scalable Discord bot systems with integrated payment gateways and real-time economies.
- ⚡ **Full Stack:** Developing high-performance web hubs and 3D interactive interfaces.
